## ✨ الميزات الرئيسية

### 🧠 ذكاء اصطناعي متقدم
- **تحليل الاتجاه الذكي**: خوارزميات متطورة لكشف اتجاهات السوق
- **إدارة مخاطر تكيفية**: نظام حماية يتأقلم مع ظروف السوق
- **معالجة الإشارات المتعددة**: دمج ذكي لعدة مؤشرات فنية

### 📊 استراتيجيات متنوعة
1. **السكالبينج** (M1-M5): للأرباح السريعة
2. **التداول المتأرجح** (H1-H4): للاتجاهات متوسطة المدى
3. **التداول المختلط**: جمع بين الاستراتيجيتين

### 🛡️ حماية شاملة
- حماية من الانسحاب الزائد
- وقف خسارة متحرك ذكي
- إغلاق جزئي للأرباح
- فلاتر متقدمة للوقت والأخبار
